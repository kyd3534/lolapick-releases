# LoLaPick Desktop

LoL 매치업 분석 + 룬 자동 적용(LCU) 데스크톱 앱 설치파일 배포용 리포입니다.
소스 코드는 비공개이며, 이곳에는 릴리즈 바이너리만 게시됩니다.

- 웹: https://lolapick.com
- 다운로드: https://lolapick.com/desktop (또는 이 리포의 Releases)

## 설치

- **macOS**: `LoLaPick-mac.dmg` — Developer ID 서명·공증 완료 (Apple Silicon + Intel 유니버설).
- **Windows**: `LoLaPick-win-setup.exe` — 코드 서명 준비 중이라 SmartScreen 경고가 보일 수 있습니다. "추가 정보 → 실행"으로 설치하세요.
